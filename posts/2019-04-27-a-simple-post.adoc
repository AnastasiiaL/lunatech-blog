The First Chapter
-----------------
Chapters can contain sub-sections nested up to three deep.
footnote:[An example footnote.]
indexterm:[Example index entry]

Chapters can have their own bibliography, glossary and index.

And now for something completely different: ((monkeys)), lions and
tigers (Bengal and Siberian) using the alternative syntax index
entries.
(((Big cats,Lions)))
(((Big cats,Tigers,Bengal Tiger)))
(((Big cats,Tigers,Siberian Tiger)))
Note that multi-entry terms generate separate index entries.

case class Post(slug: String, title: String, date: DateTime, author: Author, mainImage: String, content: String) {
  val asciidoctor = Factory.create()

  def getFirstParagraph(): String = asciidoctor.convert(
    content,
    new java.util.HashMap[String, Object]())

  def getContent(): String = asciidoctor.convert(
      content,
      new java.util.HashMap[String, Object]())

}
